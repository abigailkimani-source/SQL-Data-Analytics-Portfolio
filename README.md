# SQL Data Analytics Portfolio

## About
Four end-to-end SQL Server analytics projects covering 
Banking, Healthcare, Sales and E-commerce — 
with 53,000+ rows of enterprise-scale data.

## Projects
1. **Banking & Finance Analytics** — 1,000 customers, 10,000 transactions, 1,000 loans
2. **Healthcare Analytics** — 1,000 patients, 5,000 admissions, 5,000 billing records
3. **Sales Analytics** — 1,000 customers, 5,000 orders, 8,000 order items
4. **E-commerce Analytics** — 1,000 customers, 5,000 orders, 3,000 reviews

## Tools Used
- SQL Server (SSMS)
- T-SQL — CTEs, Subqueries, JOINs, Window Functions
- Power BI Desktop
- GitHub
